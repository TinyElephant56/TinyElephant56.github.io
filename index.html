programming projects

AutoScout

scouting is an part within robotics competitions where you record data on other teams in order to form strategies and know who is the best to pick on your alliance. this program automates that process by utilizing multiple camera angles from the event livestream.

made in python. process videos with cv2. it uses a yolov8 model trained with this dataset that i made with roboflow. it also uses open-ocr to detect numbers. yt-dlp to download youtube vids. gui made with tkinter

here is the github link (although i haven't tested installation so gl with that!)
